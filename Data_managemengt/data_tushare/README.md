# Tushare 沪深 A 股数据集说明

本目录数据由 `get_stock_data_tushare.py` 基于 Tushare 接口抓取并整理生成。

当前版本已经从原来的“沪市 A 股”口径升级为 **沪深 A 股** 口径：同时覆盖上交所 `SSE` 与深交所 `SZSE`，并排除 B 股代码段。

## 一句话结论

这份数据适合交付为 **沪深 A 股日频行情与多源特征数据集**。

如果只是做全 A 股行情建模、特征工程或排序训练，建议优先使用：

```text
stock_data.csv
```

如果需要更多扩展字段，例如估值、复权因子、资金流、融资融券、龙虎榜、涨跌停、北向持股、沪深300权重等，建议使用：

```text
stock_data_tushare_tradable.csv
```

## 字段概览

### `stock_data.csv`

该文件用于兼容现有训练流程，字段较少但最稳定：

```text

日期
开盘
收盘
最高
最低
成交量
成交额
振幅
涨跌额
换手率
涨跌幅
```

### `stock_data_tushare_tradable.csv`

该文件包含更多可用于特征工程的字段，主要类别包括：

- 基础行情：`开盘`、`收盘`、`最高`、`最低`、`pre_close`、`涨跌幅`、`成交量`、`成交额`
- 估值与市值：`pe_ttm`、`pb`、`ps_ttm`、`total_mv`、`circ_mv`
- 复权与收益：`adj_factor`、`hfq_open`、`hfq_close`、`log_return_hfq`
- 技术指标：`macd_dif`、`rsi_6`、`rsi_12`、`rsi_24`、`kdj_k`、`kdj_d`
- 资金流：`buy_el`、`sell_el`、`buy_lg`、`sell_lg`、`buy_sm`、`sell_sm`
- 融资融券：`rzye`、`rqye`、`rzjme`
- 涨跌停：`limit_status`、`fd_amount`、`first_time`、`last_time`
- 龙虎榜与机构榜：`reason`、`l_buy`、`l_sell`、`top_inst_buy_amt`、`top_inst_sell_amt`
- 北向与沪深股通：`hsgt_top10`、`hk_hold` 相关字段
- 状态标记：`is_st`、`is_tradable`
- 沪深300辅助字段：`hs300_weight`、`is_hs300_member`

## 沪深300字段说明

数据库中 `index_weight_hs300` 表覆盖：

| 项目 | 数值 |
| --- | ---: |
| 记录数 | `17,400` |
| 权重截面数 | `58` |
| 覆盖日期 | `2024-01-02` 至 `2026-05-29` |

注意：`hs300_weight` 与 `is_hs300_member` 是根据 Tushare `index_weight` 历史权重做向前匹配得到的辅助字段。

当前该字段适合用于参考沪深300历史权重，但 **不建议直接作为严格沪深300候选池过滤字段**。原因是部分股票调出沪深300后，历史权重向前匹配可能继续保留旧权重，导致某些日期 `is_hs300_member=1` 的股票数超过 300。

实测示例：

| 日期 | `is_hs300_member=1` 数量 |
| --- | ---: |
| `2024-01-02` | `300` |
| `2025-07-31` | `335` |
| `2025-08-01` | `335` |
| `2026-03-13` | `346` |
| `2026-05-29` | `346` |

如果交付目标是 **全沪深 A 股行情/特征数据集**，可以忽略这一限制。

如果交付目标是 **严格沪深300成分股回测或沪深300内选股**，建议使用专门的逐日成分股日历进行过滤，或后续单独修正 `is_hs300_member` 字段。

## 推荐使用方式

### 1. 直接用于现有训练代码

推荐使用：

```text
stock_data.csv
```

该文件字段与旧训练流程兼容，适合快速训练全沪深 A 股排序模型。

### 2. 用于增强特征训练

推荐使用：

```text
stock_data_tushare_tradable.csv
```

使用前建议：

- 删除明显不需要的高缺失事件型字段。
- 对龙虎榜、涨跌停、北向持股等事件型字段单独转成 0/1 特征。
- 不要直接把所有高缺失字段简单填 0 后全部喂给模型，除非已经验证有效。

### 3. 用于研究停牌/不可交易状态

推荐使用：

```text
stock_data_tushare_wide.csv
```

该文件保留完整股票-交易日网格，因此会包含不可交易或无行情的记录。

## 重新生成方式

在项目根目录运行：

```powershell
cd D:\bisai\test01\THU-BDC2026

$env:TUSHARE_TOKEN='你的 Tushare Token'
$env:PYTHONDONTWRITEBYTECODE='1'

python .\get_stock_data_tushare.py --reset --skip-financial --sleep 0.8
```

如果不想覆盖当前 `data_tushare`，可以指定新目录：

```powershell
python .\get_stock_data_tushare.py --output-dir data_tushare_a --reset --skip-financial --sleep 0.8
```

## 本版本限制

1. 本次运行使用了 `--skip-financial`，因此没有生成三张财务报表、财务指标、业绩预告/快报等低频财务字段。
2. 当前数据不包含北交所。
3. `stock_data_tushare_tradable.csv` 和 `stock_data_tushare_wide.csv` 文件较大，读取时建议使用 pandas 分块读取或确保内存充足。
4. `is_hs300_member` 是辅助字段，不建议作为严格沪深300逐日成分股过滤依据。
